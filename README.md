# ProspectOS
### La prospection commerciale, élevée au rang d'art.

---

**ProspectOS** est une plateforme SaaS de prospection B2B de nouvelle génération qui consolide l'intégralité du cycle de vente en un écosystème unique et harmonieux. Inspirée par une esthétique littéraire et chaleureuse, elle remplace la froideur des outils traditionnels par une expérience utilisateur réfléchie, humaine et hautement performante.

## L'Essence de la Plateforme

ProspectOS ne se contente pas d'envoyer des courriels. Il orchestre une symphonie de données et d'intelligence artificielle pour transformer chaque interaction en une opportunité authentique.

### Le Prospect Audit Engine™
Notre moteur exclusif analyse en profondeur la maturité numérique de vos prospects. En scrutant la rapidité des sites, la qualité du design, la structure SEO et la cohérence de marque, ProspectOS génère un "Opportunity Brief" qui alimente une personnalisation IA d'une pertinence inégalée.

## 🖋️ L'Art du Détail

Le design de ProspectOS suit une philosophie stricte intitulée **Parchment & Terracotta**. Chaque pixel est pensé pour évoquer le confort d'un papier de haute qualité plutôt que la froideur d'un écran.

- **Canvas** : Ton parchemin (#f5f4ed) pour une lecture reposante.
- **Accent** : Terre cuite (#c96442) pour une chaleur organique.
- **Typographie** : Anthropic Serif pour la noblesse des titres, Anthropic Sans pour la clarté de l'interface.

## 🛠️ Mise en Œuvre

### Prérequis
- Docker & Docker Compose
- Node.js & npm (pour le développement frontend)
- Python 3.11+ (pour le backend)

### Lancement Rapide
```bash
# Cloner le dépôt
git clone https://github.com/Endsi3g/uprising-outreach.git

# Initialiser l'environnement complet via Docker
make dev

# Migrer la base de données
make migrate
```

## 📐 Architecture

Le système repose sur une pile technologique robuste et moderne :
- **Frontend** : Next.js 15, TypeScript, Vanilla CSS.
- **Backend** : FastAPI (Python), PostgreSQL, Redis.
- **Orchestration** : Docker Compose, Makefile scripts.
- **Tests** : Playwright (E2E), Vitest (Frontend), Pytest (Backend).
